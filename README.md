## ЁЯЫбя╕П Safety Features

### Blocked Commands

```
тЭМ Filesystem:  rm -rf /, rm -rf ~, mkfs, dd
тЭМ SQL:         DROP TABLE, DELETE FROM, UPDATE
тЭМ System:      shutdown, reboot, halt, poweroff
тЭМ Code:        shutil.rmtree, os.remove
```

### Encoding Bypass Detection

Xalgorix detects obfuscated commands:

| Technique | Example |
|----------|--------|
| Base64 | `echo cm0gL3JmIC8= \| base64 -d` |
| Hex | `\x72\x6d\x20\x2d\x72\x66` |
| URL | `%72%6d%20%2d%72%66` |

### Circuit Breaker

After **5 consecutive failures**, a tool is temporarily blocked for **60 seconds** to prevent wasting time.

## ЁЯУЛ 22-Phase Methodology

1. ЁЯФН **Recon** тАФ Subdomains, ports, directories
2. ЁЯжа **Vuln Scan** тАФ Nuclei, nmap scripts
3. ЁЯУВ **Content** тАФ Fuzzing, backups, admin panels
4. ЁЯФР **SSL/TLS** тАФ Cipher, certificates, headers
5. ЁЯФС **Auth** тАФ SQLi login, brute-force, OAuth
6. ЁЯТЙ **Injection** тАФ XSS, SQLi, Command, XXE, SSTI
